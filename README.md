# ProyectoMoviles
Kevin Vera Fernandez

Descripción General del Proyecto

- KCH es una aplicación móvil desarrollada en Android Studio que funcionara como puente virtual entre pacientes y doctores, mediante esta se podrá solicitar citas medicas personalizadas, gestionar su historial de médicos, recibir consejos de salud, entre otras funciones útiles. La aplicación tendrá una interfaz intuitiva y atractiva que brindará una experiencia similar y mejorada con respecto a otras las apps de esta índole.


Objetivo General

- Desarrollar una aplicación móvil que facilite a los pacientes estar al día y llevar un control de su salud, la selección de médicos, la gestión de solicitudes y el almacenamiento de sus preferencias.


Objetivos Específicos

- Diseñar una interfaz responsiva y moderna que a su vez sea amigable con cada tipo de usuario.
- Implementar un CRUD completo (Crear, Leer, Actualizar y Eliminar) para la gestión de médicos, citas, entre otros.
- Implementar Dialogs para confirmar acciones (p,ej: eliminar un historial, confirmar elección de doctor o mostrar información adicional).\
- Incorporar la funcionalidad para agregar imágenes de los medicamentos, recetas o examenes desde la galería o cámara.
- Utilizar listas personalizadas con capacidad de búsqueda y filtrado por tipo de necesidad.
